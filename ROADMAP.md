# Haikode Roadmap

## MVP

- Native app shell.
- Open project.
- File tree and read-only file viewer.
- Project scanner with ignore rules.
- `.haikode/project.json` local metadata.
- Optional BFS attributes.
- OpenAI-compatible AI chat with selected file context.
- Haiku-aware prompt builder.
- Unified diff generation and raw diff preview.
- Patch save/apply/reject with backups.
- Build/test command runner with approval.
- Command output logs.
- README with build, configuration, safety, and limitations.

## Post-MVP 1

- Native settings window.
- Better diff UI with per-file/per-hunk review.
- Git status panel.
- Live BFS query project radar.
- More complete JSON parser/writer.
- Restore-from-backup action.
- Session history browser.

## Post-MVP 2

- Haiku app template.
- Add window/menu/file picker helpers.
- HPKG packaging checklist.
- HaikuPorts recipe helper.
- Build error auto-diagnosis.
- Local model support.

## Post-MVP 3

- Plugin system.
- Project radar dashboard.
- Tracker integration.
- File attribute search.
- Visual asset preview.
- Git commit and PR helpers.
